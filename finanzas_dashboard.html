Configuración inicial

Para conectar la app con tu Google Sheet necesitas un Client ID de OAuth 2.0. Sigue los pasos a continuación. Solo necesitas hacerlo una vez.

  1. Ve a console.cloud.google.com y crea un nuevo proyecto.
  2. En el menú: APIs y Servicios → Biblioteca → busca Google Sheets API → Habilitar.
  3. Ve a APIs y Servicios → Credenciales → Crear credenciales → ID de cliente OAuth 2.0.
  4. Tipo de aplicación: Aplicación web.
  5. En Orígenes de JavaScript autorizados, agrega: http://localhost y el dominio donde alojes este archivo.
  6. Copia el ID de cliente generado y pégalo abajo.
